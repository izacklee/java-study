#### 3 redis
   5种数据类型：string, hash, list, set, zset
   是什么？基于内存运行的分布式的，性能高效，高并发读写，持久化，单线程的中间件。
   干什么用？缓存，计数器，分布式锁。
   可以解决什么问题？高并发问题。
   为什么redis单线程速度还这么快？因为全在内存进行，不涉及任何IO，所以速度很快。
   redis是单线程的吗？命令执行是单线程，网络连接（redis6.0 属于IO操作）是多线程，大的key的删除是多线程（redis4.0）,
   未来会在更多的地方引入多线程。
